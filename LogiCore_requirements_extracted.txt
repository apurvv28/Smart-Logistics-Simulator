Tech  Stack  (Java/C++)
 ●  Core  Logic:  Java  (Spring  Boot  3.x)  OR  C++  (Crow  Framework) .  ●  Data  Structures:  Custom-built  Adjacency  Lists,  Priority  Queues  (Min-Heaps),

 ●  Database:  MySQL  +  PostGIS  for  spatial  data  storage  and  geographic

 ●  Frontend:  React.js  +  Tailwind  CSS  with  Cytoscape.js  for  interactive  graph
